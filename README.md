#Asistente virtual KAI 

es un proyecto sencillo de asistente por consola desarrollado en Python, pensado como práctica de programación y como base para futuros bots más avanzados.

**Descripción**

KAI se ejecuta en la terminal y mantiene una conversación básica con el usuario, respondiendo a comandos o preguntas predefinidas. El objetivo del proyecto es ejercitar:

Manejo de entrada y salida por consola.

Estructuras de control (condicionales y bucles).

Organización del código en funciones.

Diseño de un flujo conversacional simple.

La lógica está escrita en un único archivo principal (Kai.py), lo que facilita entender el funcionamiento y modificarlo.

**Uso básico**

Crear y activar un entorno virtual (opcional pero recomendado).

Ejecutar el script:

```
python Kai.py
```
Escribir los comandos que el asistente reconoce (por ejemplo, saludos, peticiones de ayuda, salida del programa, etc.).

KAI está pensado para que sea fácil agregar nuevos comandos y respuestas personalizadas, por lo que es un buen punto de partida para aprender a construir asistentes más completos o integrar futuras funcionalidades (APIs, archivos de configuración, menús, etc.).
